# Calora

Мобильное приложение для подсчёта калорий, отслеживания КБЖУ и микронутриентов, учёта тренировок.

## Описание

Calora помогает пользователю контролировать питание: вести дневник приёмов пищи, рассчитывать дневную норму КБЖУ на основе персональных параметров, отслеживать микронутриенты и учитывать сожжённые калории от упражнений.

**Ключевая особенность** — AI-чат для добавления еды. Вместо поиска по базам достаточно описать блюдо словами (например: «варёные яйца 2 шт» или «тарелка борща со сметаной»), и AI рассчитает калории, БЖУ и микронутриенты.

## Технологический стек

- **React Native** (0.81) + **Expo** SDK 54
- **TypeScript** (~5.9)
- **React Navigation** — Tab + Stack навигация с анимациями
- **AsyncStorage** — локальное хранилище данных (offline-first, без регистрации)
- **react-native-svg** — графики и прогресс-кольца
- **react-native-chart-kit** — графики аналитики
- **expo-camera** — сканер штрих-кодов
- **expo-haptics** — тактильная обратная связь
- **expo-notifications** — напоминания (development build)
- **expo-file-system** + **expo-sharing** — экспорт/импорт данных
- **Groq API** (Llama 3.3 70B) — AI-анализ продуктов через Vercel-прокси

## Архитектура AI

```
┌──────────────┐     POST /api/ai-nutrition     ┌──────────────────┐     Groq API     ┌────────┐
│  Mobile App  │  ─────────────────────────────> │  Vercel Proxy    │ ───────────────> │  Groq  │
│  (Expo)      │ <───────────────────────────── │  (CaloraAI repo) │ <─────────────── │  LLM   │
└──────────────┘     { analysis: {...} }         └──────────────────┘                  └────────┘
```

- API-ключ хранится **только на сервере** (Vercel env), не попадает в мобильный бандл
- Прокси: rate limiting (20 req/min), валидация, CORS
- Клиент: лимит 10 запросов/день на устройство, счётчик с цветовой индикацией
- Репозиторий прокси: [CaloraAI](https://github.com/DarkBeer0/CaloraAI)

## Реализованные функции

### AI-чат добавления еды
- Описание блюда словами → карточка с калориями, БЖУ, пользой
- Редактирование граммов прямо на карточке до добавления
- Добавление в рацион одной кнопкой (тип приёма по времени суток)
- Лимит 10 запросов/день со счётчиком (зелёный → оранжевый → красный)
- Примеры-подсказки на экране приветствия

### Профиль и расчёт нормы
- Возраст, вес, рост, пол, уровень активности, цель
- Живой пересчёт КБЖУ (Mifflin-St Jeor + весовые коэффициенты белка)
- Настраиваемая цель воды

### Дашборд
- Анимированное кольцо калорий (потреблено / сожжено / осталось)
- 3 мини-кольца БЖУ
- Микронутриенты: клетчатка, сахар, насыщ. жиры, соль
- Трекер воды
- Список упражнений и приёмов пищи
- Long-press на карточке еды → редактировать / удалить
- Pull-to-refresh, staggered fade-in анимации

### История и аналитика
- История по дням с копированием на сегодня
- Графики калорий за неделю/месяц
- Распределение макросов (pie chart)
- Трекинг веса с графиком
- Экспорт/импорт данных (JSON)

### Упражнения
- 10 типов с MET-расчётом
- Swipe-to-delete, haptic feedback

### Штрих-код и ручной ввод
- Сканер через expo-camera
- Ручной ввод всех 8 нутриентов
- Open Food Facts как fallback

### Рецепты
- Составные блюда из нескольких ингредиентов
- Расчёт на порцию

### Прочее
- Тёмная/светлая тема + сохранение выбора
- 3 языка: русский, английский, польский
- Уведомления (development build): еда, вода, итоги дня
- Онбординг при первом запуске
- ErrorBoundary для устойчивости
- Анимации переходов между экранами
