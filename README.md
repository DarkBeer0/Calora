# Calora

Мобильное приложение для подсчёта калорий, отслеживания КБЖУ и микронутриентов, учёта тренировок.

## Описание

Calora помогает пользователю контролировать питание: вести дневник приёмов пищи, рассчитывать дневную норму КБЖУ на основе персональных параметров, отслеживать микронутриенты и учитывать сожжённые калории от упражнений.

## Технологический стек

- **React Native** (0.81) + **Expo** SDK 54
- **TypeScript** (~5.9)
- **React Navigation** — Tab + Stack навигация
- **AsyncStorage** — локальное хранилище данных
- **react-native-svg** — графики и прогресс-кольца
- **expo-camera** — сканер штрих-кодов
- **Open Food Facts API** — база продуктов питания
- **Supabase** — бэкенд *(планируется)*

## Реализованные функции

### Фаза 1 — Навигация (готово)
- Tab-навигация: Главная / История / Профиль с иконками
- Stack-навигация: 5 модальных экранов (AddMeal, ConfirmMeal, AddExercise, BarcodeScanner, AddCustomFood)
- Поддержка тёмной темы в навигации

### Фаза 2 — Профиль (готово)
- Форма ввода: возраст, вес, рост, пол, уровень активности (5 вариантов), цель (3 варианта)
- Живой пересчёт нормы КБЖУ при каждом изменении любого поля
- Сохранение в AsyncStorage

### Фаза 3 — Дашборд (готово)
- CalorieSummaryCard: большое кольцо с «осталось ккал», «потреблено» и «сожжено»
- 3 мини-кольца БЖУ с подписями «ещё Xг»
- Секция микронутриентов: клетчатка, сахар, насыщ. жиры, соль (с предупреждением при превышении)
- Секция упражнений с кнопкой «Добавить»
- Список приёмов пищи по типам (завтрак/обед/ужин/перекус)
- FAB-кнопка «+» для добавления еды

### Фаза 4 — Добавление приёма пищи (готово)
- Текстовый поиск через Open Food Facts API
- Кнопки быстрого действия: «Штрих-код» и «Вручную»
- Экран подтверждения: ввод граммовки, быстрый выбор (50/100/150/200/300г)
- Превью всех 8 нутриентов перед сохранением
- Выбор типа приёма пищи

### Фаза 5 — История (готово)
- Список дней с итогами КБЖУ
- Раскрывающиеся карточки с детализацией по типам
- Удаление записей
- «Сегодня» / «Вчера» вместо дат

### Фаза 7 — Штрих-код, кастомные продукты, тёмная тема (готово)
- Сканер штрих-кодов через expo-camera (EAN-13, EAN-8, UPC)
- Если продукт не найден — предложение добавить вручную
- Экран ручного ввода продукта (все 8 нутриентов)
- Тёмная тема с полной палитрой + сохранение выбора
- Переключатель темы на экране профиля

### Упражнения (готово)
- 10 типов тренировок с иконками (ходьба, бег, велосипед, плавание, зал, йога, HIIT и др.)
- Расчёт сожжённых калорий по MET-формуле
- Быстрый выбор длительности (15/30/45/60/90 мин)
- Сожжённые калории вычитаются из баланса на дашборде

### Микронутриенты (готово)
- Клетчатка (норма 25г, чем больше — тем лучше)
- Сахар (лимит 50г, превышение = красный)
- Насыщенные жиры (лимит 20г)
- Соль (лимит 5г)

## Запуск

```bash
npm install
npx expo start
```

Отсканируйте QR-код в приложении **Expo Go** на телефоне.

## Переменные окружения

Скопировать `.env.example` в `.env` и заполнить (нужно только для Фазы 6):

```
EXPO_PUBLIC_SUPABASE_URL=your-supabase-url
EXPO_PUBLIC_SUPABASE_ANON_KEY=your-supabase-anon-key
```
